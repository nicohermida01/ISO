# Comandos utilizados en la practica 3

- `who`: Muestra los usuarios actualmente conectados al sistema.

- `wc`: Cuenta líneas, palabras y bytes en archivos o entrada estándar.

- `grep`: Busca patrones en archivos o entrada estándar y muestra las líneas que coinciden.

- `find`: Busca archivos y directorios en una jerarquía de directorios según criterios especificados.

- `sleep`: Pausa la ejecución de un script o comando por un período de tiempo especificado.

- `tr`: Transforma o elimina caracteres de la entrada estándar.

- `basename`: Extrae el nombre del archivo de una ruta completa.

- `dirname`: Extrae el directorio de una ruta completa.

- `cut`: Extrae secciones de cada línea de entrada, como campos o caracteres específicos.

- `unset`: Elimina variables o funciones del entorno del shell.

- `getent`: Recupera entradas de bases de datos del sistema, como usuarios o grupos.

- `awk`: Un lenguaje de programación para procesamiento de texto y generación de informes, útil para manipular y analizar datos en archivos de texto.
